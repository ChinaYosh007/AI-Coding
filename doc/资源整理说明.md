# 后端资源整理说明

本次文档只归档接口说明和数据库 SQL；课程笔记不上传到 `doc`，继续保留在后端资源目录。

## 已归档

| 文档资源 | 位置 | 处理方式 |
| --- | --- | --- |
| 接口说明 | `doc/接口文档.md` | 根据 Controller、Swagger 与前端生成客户端整理 |
| 数据库初始化脚本 | `doc/sql/init.sql` | 从后端原脚本复制并校验 SHA-256 一致 |
| SQL 使用说明 | `doc/sql/README.md` | 说明导入方式和原脚本位置 |

## 保留在后端资源目录

| 路径 | 原因 |
| --- | --- |
| `static/docs/` | 课程学习笔记，不纳入项目交付文档 |
| `static/sql/init.sql` | 后端原始初始化资源；本次不修改、不移动 |
| `static/template/` | 生成 Vue 项目的运行模板 |
| `prompt/` | AI 代码生成与修改提示词 |
| `mapper/` | MyBatis Mapper XML |
| `static/request/test.http` | 开发调试请求文件 |
| `static/tmp/` | 现存生成产物；未确认依赖关系前不删除 |

后续如需清理 `static/tmp/`，应先确认没有应用版本 `sourcePath`、预览或部署流程依赖其中内容，再进行单独清理。
